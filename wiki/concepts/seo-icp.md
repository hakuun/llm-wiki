# SEO ICP

SEO ICP is the ideal customer profile for search: the person typing a business-relevant query into a search box. For AI tool sites, it focuses less on demographic persona and more on the searcher's current task, intent, evidence needs, and conversion path.

## Practical Dimensions

Use these fields when planning pages:

- Role, industry, and region.
- Task the searcher wants to complete.
- Intent stage: informational, commercial, transactional, or navigational.
- Keyword cluster: head terms, long-tail terms, synonyms, and question terms.
- Evidence preference: price, privacy, quality, benchmark, screenshot, API detail, or comparison.
- Conversion path: demo, registration, trial, payment, download, or API docs.

## AI Tool Site Roles

The source highlights three common search users:

- **User** — wants an immediately usable feature, template, generator, or prompt.
- **Buyer** — compares tools, pricing, APIs, and alternatives before adoption.
- **Learner** — wants to understand a concept, workflow, or best practice.

## Use in SEO Execution

SEO ICP should determine:

- Which keyword clusters are worth pursuing.
- Which page type fits the query.
- Which content modules are required.
- Which CTA should appear on the page.
- Which evidence will help the user stop searching.
